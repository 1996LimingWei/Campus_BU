# 香港浸会大学留位费缴纳指南 (2026 Fall)

> 整理日期: 2026-04-20
> 唯一品牌: 香港浸会大学 (HKBU)

## 一、缴费流程概览

### 1. 查看相关文件

登录网申系统，点击 **Application Details** -> **Application Status**，下载以下文件:

- **Admission Offer Letter** (录取信)
- **Note 1: Procedures for Acceptance of Offer** (Offer接受步骤)
- **Note 2: Preparation for Registration as an HKBU Student** (HKBU学生注册准备)

### 2. 查看留位费详情

点击 **Debit Note Details** 查看留位费金额和缴费方式说明。

缴费时需提供:
- **Debit Note No.** (14位账单号) — 飞汇支付或在香港缴费时使用
- **Student No.** (8位学生号，非申请号) — 内地生电汇和汇票时使用

点击 Payment Method 后的蓝色字体下载付款方式指导文件。

### 3. 上传缴费凭证

缴费后，须在录取信上列明的 **Due Date** 当天或之前:
1. 登录网申系统
2. 点击 **Accept Offer**
3. 勾选 Step 1 三个条款
4. 上传缴费凭证
5. 点击下方 **Accept Offer** 正式接受录取

缴费后约 1-2 周学校会在 Payment Status 更新状态。

## 二、缴费方式详解

### 方式一: Flywire 飞汇支付平台

- **支持方式**: 银行转账、支付宝、中国银联、网上银行、信用卡
- **手续费**: 无
- **到账时间**: 2-3个工作日
- **登录地址**: https://landing-pages.flywire.com/landing/hkbu
- **操作**: 选择 Tuition Fee (exclude SCE) -> Start Your Payment，按指示付款
- **详细教程**: https://payment.flywire.com/pay/payment

### 方式二: 电汇 (Telegraphic Transfer)

需在留位费基础上多付 **300港币手续费**。

#### 网银/手机银行电汇

1. 确保银行卡有多币种功能，已开通网银
2. 先进行购汇（港币）
3. 进入"转账汇款" -> "境外汇款"
4. 填写收款人信息（参考 Offer 信中的说明）
5. 汇款资金用途: 选择 **留学学费（一年以上）**
6. 附言: **学生编号 + 姓名拼音**

#### 银行柜台电汇

1. 提前致电银行确认可操作
2. 携带: 身份证、银行卡或现金、Offer复印件
3. 告知工作人员"电汇港币"
4. 汇款币种: **HKD**，收款人: 学校英文全称
5. 交易附言: **学生编号 + 姓名拼音**

**电汇后**: 上传电汇凭证回单到网申系统，并电邮汇款单至 **fostudent@hkbu.edu.hk** 以便核实。

### 方式三: 汇票

1. 提前致电银行确认可当天出票
2. 携带: 身份证原件、银行卡或现金、Offer复印件
3. 收款人名称: 学校英文全称
4. 汇票背面写上: **学生编号 + 姓名拼音 + 学生号**
5. 邮寄汇票给学校
6. 保存"境外汇款申请书"作为凭证
7. 寄出后发邮件通知学校，附汇票照片

**汇票丢失处理**: 带证件和"境外汇款申请书"去银行挂失。

### 方式四: 在港缴费

适用于本人在香港或拥有香港本地银行账户:
- 信用卡
- 银行自动柜员机
- 缴费灵
- 网上银行
- 现金或支票于恒生银行柜台

具体操作步骤参考学校下发文件。

## 三、常见问题

**Q: 最晚什么时候汇留位费？**
A: 至少提前2天，外汇有时会被抽查，流程可能较慢。

**Q: 线上/网银付款失败怎么办？**
A: 可能原因:
1. 浏览器版本问题（建议用Chrome）
2. 未开通国际线上支付功能
3. 银行卡设有支付限额（联系银行调整）
4. 余额/信用额度不足
5. 超过个人境外汇款额度

**Q: 留位费能退吗？**
A: 原则上不能退。如放弃录取，直接与学校沟通。

**Q: 必须用本人的卡吗？**
A: 可以使用他人卡代付。

**Q: 留位费付款成功后怎么处理？**
A: 录取资格保留，入学后留位费直接转成学费。

**Q: 如何确认付款成功？**
A: 在线支付可登录网申系统查看；电汇可发邮件询问学校。

**Q: 电汇到账速度？**
A: 与汇款币种、银行工作日、收款人信息准确性有关。通常1-2个工作日到账。

**Q: 收到的款项比汇出金额少？**
A: 转汇行会扣收转汇费，属正常现象。

> 以上信息仅供参考，请以学校发布的官方文件为准。
